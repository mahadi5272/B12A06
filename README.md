## 1) What is the difference between var, let, and const?
 Here’s a concise answer in English:
A:  var: Function-scoped, can be redeclared and updated.

B: let: Block-scoped, can be updated but cannot be redeclared in the same scope.

C: const: Block-scoped, cannot be redeclared or updated (the value is constant).

# Courier Project

![Project Screenshot](https://via.placeholder.com/800x400.png?text=Courier+Project+Screenshot)

## Overview
Courier Project একটি ওয়েব অ্যাপ্লিকেশন যা কুরিয়ার ডেলিভারি সিস্টেমকে সহজ এবং ইন্টারেকটিভ করে।  
ইউজাররা সহজে কুরিয়ার অর্ডার করতে পারবে, ডেলিভারি ট্র্যাক করতে পারবে এবং ডেলিভারি স্ট্যাটাস দেখতে পারবে।  

## Main Features
- ইউজার রেজিস্ট্রেশন এবং লগইন (Local Storage ব্যবহার করে)
- কুরিয়ার অর্ডার ফর্ম এবং ডাইনামিক ডেটা ভ্যালিডেশন
- অর্ডার ট্র্যাকিং সিস্টেম
- রেসপন্সিভ ডিজাইন, সকল ডিভাইসে ব্যবহারযোগ্য
- ক্লিন এবং মডার্ন ইউজার ইন্টারফেস

## Technologies Used
- **HTML5** – ওয়েবপেজের স্ট্রাকচার  
- **CSS3** – স্টাইলিং এবং লেআউট  
- **JavaScript** – ডাইনামিক ফাংশনালিটি এবং ভ্যালিডেশন  

## Dependencies
- কোনো এক্সটার্নাল লাইব্রেরি ব্যবহার করা হয়নি।  
- সকল ফাংশনালিটি Vanilla JS, HTML, এবং CSS দিয়ে তৈরি।
